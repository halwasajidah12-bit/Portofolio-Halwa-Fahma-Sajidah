# Portofolio-Halwa-Fahma-Sajidah
website Portofolio sederhana dengan menggunakan bahasa pemograman html, css, dan javascript. Terdapat 5 halaman pada website 
Project mandiri Membuat portofolio sederhana Nama: Halwa Fahma Sajidah NIM: 24051204037 Penilaian tengah semester projek membuat portofolio sederhana dengan menggunakan bahasa pemograman html, css dan javascript dan bootsrap untuk framework css. Portofolio ini berisi 5 halaman: halaman pertama berupa home page halaman kedua berisi Deskripsi diri halaman ketiga berisi porto atau pun karya atau pengalaman berupa foto" dan juga hasil vidio editing halaman keempat berisi kumpulan certificate yang dapat diakses melalui home page halaman ke lima berisi kontak pribadi berupa akun instagram dan juga whatsapp
